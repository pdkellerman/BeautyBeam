# BeautyBeam
A colorful gradient icon theme for KDE Plasma, designed for dark themes.

KDE Plasma Only!

This icon pack is designed specifically for KDE Plasma.

My own contributions to this icon pack are released under GPL-3.0-or-later. Third-party icons, artwork, and assets remain under their original licenses. Please see the included CREDITS.md file for full attribution, license details, and source information.

Before anything else, I want to make it clear that the original icon artwork is not my own. My work mainly involved recoloring, editing, combining, and adapting icons into a more colorful gradient-based style.

Full credit goes to the original artists and icon pack creators. This pack uses and references icons from many sources, including BeautyLine, Candy Icons, Cawnonical Mono-Glyph, Gruvbox Plus, MingCute Icons, Papirus, Solar Icons, Sours, and many others.

There are too many sources to list properly here, so please check the included Credits file for full attribution and support the original authors wherever possible.

The goal of this icon pack is to make KDE Plasma feel more colorful, vibrant, and gradient-heavy, especially when used with a dark theme. Personally, I use it with Sweet KDE by EliverLara:
https://store.kde.org/p/1294174/

This is my first attempt at creating a full icon pack, so please be kind. In general, KDE Plasma, Inkscape, Dolphin, and Konsole look good with it, although I have not yet tested every application thoroughly.

I will keep improving the pack over time.
